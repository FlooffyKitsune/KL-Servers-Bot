Kitty League Server Management Bot


Terms of Service

KL Servers Bot is a bot purpose built for the management of Kitty League game servers. KL Servers Bot is for use by members of Kitty League servers and partners of Kitty League. If you are not a member or partner, by using KL Servers Bot, you recognize the bot will not function on/for your server. However, source code may be forked and repurposed for your own server and use cases.

You are responsible for upholding Discord's ToS: https://discord.com/terms.

Privacy Policy

Data Collected

When utilizing KL Servers Bot, the Discord application collects the following data:

  • Discord User ID
  • Discord Username
  • Command Used
  • Timestamp of Utilization

Message Content

KL Servers Bot is able to read the content of messages sent in your Discord server, however no messages are stored. Only commands specific to the bot are stored for the following reasons:

  • Debugging
  • Verifying Bot Functionality
  • Identity Verification
  • Logging
